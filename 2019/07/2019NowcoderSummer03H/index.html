「2019牛客暑期多校 - 第三场」H - Magic Line
2019-07-30 13:20   folder 2019牛客暑期多校训练营 - 第三场 label 数学, 几何, 平面直角坐标系

题目传送门

_Say

求一条线把给定的N个点平均分成左右两各部分，输出这条直线上任意两点的坐标。
最简单的做法就是把这些点按横坐标从小到大的顺序排序，如果横坐标相等，就按纵坐标从小到大排序。然后选定中间偏左的点，在这个点右边一点点的地方接近竖直地画一条线，即为所求。

_Code

#include<bits/stdc++.h>
using namespace std;
struct point{
    int x,y;
}p[1001];
int t,n;
int main(){
    ios::sync_with_stdio(false);cin.tie(0);
    while(cin>>t){
        while(t--){
            cin>>n;
            for(int i=0;i<n;i++)cin>>p[i].x>>p[i].y;
            sort(p,p+n,[](point &a,point &b){return a.x<b.x||a.x==b.x&&a.y<b.y;});
            int m=n/2-1;
            cout<<p[m].x-1<<" "<<"999000000"<<" "<<p[m].x+1<<" "<<2*p[m].y+1-999000000<<endl;
        }
    }
    return 0;
}
