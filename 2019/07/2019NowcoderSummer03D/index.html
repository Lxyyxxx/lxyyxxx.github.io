「2019牛客暑期多校 - 第三场」D - Big Integer
2019-07-31 17:49   folder 2019牛客暑期多校训练营 - 第三场 label 数学, 因式分解, 快速幂取模, 费马小定理

题目传送门

_Say

找规律。
推公式。
就是求数列1,11,111,1111...1,11,111,1111...1,11,111,1111...中，满足第iji^jij项是ppp的倍数(i,j)(i,j)(i,j)的对数。

_Solve

这是显然的。严谨的推导证明好难，我觉得我在乱写。
首先，当p=2p=2p=2或p=5p=5p=5时，答案显然是0。

接下来考虑通解。
数列的通项公式是A(n)=10n−19A(n) = \dfrac{10^n - 1}{9}A(n)=910n−1​，那么A(n)≡0(mod p)A(n) \equiv 0 (mod \ p)A(n)≡0(mod p)就等价于10n≡1(mod 9p)10^n \equiv 1 (mod \ 9p)10n≡1(mod 9p)。因为9∣(10n−1)9|(10^n - 1)9∣(10n−1)，所以当ppp与9互质时，又可以化简成10n≡1(mod p)10^n \equiv 1 (mod \ p)10n≡1(mod p)。
因此，这里要考虑当ppp与9不互质的情况，也就是当p=3p=3p=3时，显然可以推出答案是⌊n÷3⌋×m\lfloor n \div 3 \rfloor \times m⌊n÷3⌋×m。
回到通解。
由费马小定理，有10p−1≡1(mod p)10^{p-1} \equiv 1 (mod \ p)10p−1≡1(mod p)，那么nnn就是所有p−1p-1p−1的倍数，这里把p−1p-1p−1称为循环节。
那么有没有比p−1p-1p−1更小的数可以作为循环节呢？
设这个数为ddd，显然d∣(p−1)d|(p-1)d∣(p−1)，将p−1p-1p−1进行因式分解，求出最小的ddd使10d≡1(mod p)10^d \equiv 1 (mod \ p)10d≡1(mod p) 。

然后就需要计算有多少对(i,j)(i,j)(i,j)使得iji^jij是ddd的倍数。
这里我们可以固定住jjj，对于每个jjj计算有多少个iii符合条件，然后累加起来即可。
那么要如何快速地判断有多少个iii呢？
还是采用因式分解的方式。
对ddd采用因式分解，d=∏i=1xpiaid = \displaystyle \prod_{i=1}^x p_i^{a_i}d=i=1∏x​piai​​。同时，对iii进行因式分解，i=∏i=1xpibii = \displaystyle \prod_{i=1}^x p_i^{b_i}i=i=1∏x​pibi​​。那么ij=∏i=1xpijbii^j = \displaystyle \prod_{i=1}^x p_i^{jb_i}ij=i=1∏x​pijbi​​。
显然，要满足d∣ijd|i^jd∣ij，就要对于任意iii，满足j×bi⩾aij \times b_i \geqslant a_ij×bi​⩾ai​，即bi⩾aijb_i \geqslant \dfrac{a_i}{j}bi​⩾jai​​。
设g=∏i=1xpi⌈aij⌉g = \displaystyle \prod_{i=1}^x p_i^{\lceil \dfrac{a_i}{j} \rceil}g=i=1∏x​pi⌈jai​​⌉​，显然对每个jjj，有⌊ng⌋\lfloor \dfrac{n}{g} \rfloor⌊gn​⌋个iii符合条件。

因为log⁡2109=29.90\log_{2}{10^9}=29.90log2​109=29.90，所以对于j>30j>30j>30的情况与j=30j=30j=30的情况相同，只需要枚举出1⩽j⩽301 \leqslant j \leqslant 301⩽j⩽30时的ggg计算即可。

综上，除去特殊情况后的求解步骤为：

  1. 因式分解p−1p-1p−1，找循环节ddd
  2. 因式分解ddd
  3. 枚举jjj，算ggg，求iii，累加得出结果

#include<bits/stdc++.h>
using namespace std;
// 快速幂取模
long long getpow(long long x,long long y,long long mod){ // x^y%mod
    long long t=1;
    while(y){
        if(y&1) t=t*x%mod;
        x=x*x%mod;
        y>>=1;
    }
    return t;
}
// 因式分解
long long pk[35],bk[35]; // pk因数，bk幂
int k; // k个因子
void getfactor(long long x){
    k=0;
    for(int i=2;i*i<=x;i++){
        if(x%i==0){
            pk[++k]=i;
            bk[k]=0;
            while(x%i==0){
                bk[k]++;
                x/=i;
            }
        }
    }
    if(x!=1){
        pk[++k]=x;
        bk[k]=1;
    }
}
int main(){
    ios::sync_with_stdio(false);cin.tie(0);
    int t;
    while(cin>>t){
        while(t--){
            long long p,n,m;
            cin>>p>>n>>m;
            long long count=0,d;
            if(p==2||p==5)count=0;
            else if(p==11)count=n/2*m; // 因为p=11时显然能得到答案，所以写在这里了
            else if(p==3)count=n/3*m;
            else{
                d=p-1;
                // 找循环节d
                getfactor(d);
                for(int i=1;i<=k;i++){
                    for(int j=1;j<=bk[i];j++){
                        long long x=d/pk[i];
                        if(getpow(10,x,p)==1&&d>x)d=x; // 10^d%p=1
                    }
                }
                // 因式分解d
                getfactor(d);
                // 计算count
                for(int j=1;j<=m;j++){
                    long long g=1;
                    for(int i=1;i<=k;i++)g*=getpow(pk[i],(bk[i]+j-1)/j,p); // 快速幂取模算g
                    count+=(n/g); // i的个数
                    if(j>=30){
                        count+=(m-j)*(n/g); // 直接把剩下的用j=30时的g一起算
                        break;
                    }
                }
            }
            cout<<count<<endl;
        }
    }
    return 0;
}
