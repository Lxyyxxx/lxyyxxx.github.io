# 图像的概念

一个矩阵，其元素代表空间的一个点，称之为像素（Pixel），每个像素的颜色和亮度用二进制数来表示，这种图像也称为位图
对于黑白图用1位表示，对于灰度图常用4位（16种灰度等级）或8位（256种灰度等级）来表示某一个点的亮度，而彩色图像则有多种描述方法

  1. 邻点ppp和qqq是4邻接的

2. qqq在ND(p)N_D(p)ND​(p)中，并且集合N4(p)N_4(p)N4​(p)与N4(q)N_4(q)N4​(q)相交的像素里，没有VVV值的像素

rgbPic = imread('x.jpg');

% 线性灰度变换函数
function y = linear1(x,a,b,c,d)
y = d.*(x>b) + ((d-c)/(b-a)*(x-a)+c).*(x>=a & x<=b) + c.*(x<a);
y(y>255) = 255;
y = uint8(y);
end
% 分段线性灰度变换函数
function y = linear2(x,mf,mg,a,b,c,d)
y = ((mg-d)/(mf-b)*(x-b)+d).*(x>=b & x<mf) + ((d-c)/(b-a)*(x-a)+c).*(x>=a & x<b) + (c/a*x).*(x>=0 & x<a);
y(y>255) = 255;
y = uint8(y);
end

    if i ~= 1

直方图均衡化实质上是：

  1. 直方图均衡后的灰度级数量(动态范围)并没有减少
  2. 减少非零频数的灰度级数以换取对比度的加大
  3. 在均衡过程中，原来的直方图上频数较小的灰度级被归入很少几个或一个灰度级内故得不到增强。若这些灰度级所构成的图象细节比较重要，则需采用局部区域直方图均衡

色调H由角度表示，反映了该颜色最接近的波长，一般红色在0∘0^\circ0∘，绿色在120∘120^\circ120∘，蓝色在240∘240^\circ240∘
饱和度S由原点到当前点的距离表示，距离越大，饱和度越高
亮度I定义在柱形彩色空间的高度，高度越高，。。。
