# 7-1 Prime Day (20分)

关键原文及大意：

not only that the corresponding number of the date 20190523 is a prime, but all its sub-strings ended at the last digit 3 are prime numbers.
（Prime Day的定义）不仅日期20190523是质数，且其每个以3为最后一个字符的子串都是质数。

For each given date, output in the decreasing order of the length of the substrings, each occupies a line. In each line, print the string first, followed by a space, then Yes if it is a prime number, or No if not. If this date is a Prime Day, print in the last line All Prime!.
对于给定的日期，按子串长度的递减顺序输出，每个占一行。对于每一行，先输出子串，再输出一个空格，然后输出是否是质数，若是，输出Yes，否则，输出No。如果这个日期是Prime Day，则在最后一行输出All Prime!。

# Idea

手写一个判断质数的函数，然后substr+stoi就签到成功了
开局9分钟才提交，因为理解错意思了，以为its sub-strings ended at the last digit 3的意思是最后3位组成的子串……

# Code

#include<bits/stdc++.h>
using namespace std;
bool isprime(int x){
    if(x==1)
        return false;
    for(int i=2;i*i<=x;i++)
        if(x%i==0)
            return false;
    return true;
}
bool solve(string s){
    return isprime(stoi(s)); 
}
int main(){
    string s;
    cin>>s;
    bool f=1;
    for(int i=0;i<(int)s.size();i++){
        string ss=s.substr(i,s.size());
        if(solve(ss)){
            cout<<ss<<" Yes\n";
        }else{
            cout<<ss<<" No\n";
            f=false;
        }
    }
    if(f)
        cout<<"All Prime!\n";
    return 0;
}

# 7-2 The Judger (25分)

关键原文及大意：

A game of numbers has the following rules: at the beginning, two distinct positive integers are given by the judge. Then each player in turn must give a number to the judge. The number must be the difference of two numbers that are previously given, and must not be duplicated to any of the existed numbers. The game will run for several rounds. The one who gives a duplicate number or even a wrong number will be kicked out.
一个数字游戏遵循以下规则：在最开始，评测系统给出两个不同的正整数。然后每个玩家按顺序必须给一个数字给评测系统。要求这个数字必须是两个前面给出的数字之差，且必须不同于任何已存在的数字。这个比赛将会持续多轮。给出重复数字或错误数字的玩家将被淘汰。

If the i-th player is kicked out in the k-th round, print in a line Round #k: i is out.. The rest of the numbers given by the one who is out of the game will be ignored. If more than one player is out in the same round, print them in increasing order of their indices. When the game is over, print in the last line Winner(s): W1 W2 ... Wn, where W1 … Wn are the indices of the winners in increasing order. All the numbers in a line must be separated by exactly one space, and there must be no extra space at the beginning or the end of the line. If there is no winner, print No winner. instead.
如果第iii位玩家在第kkk轮被淘汰了，输出一行Round #k: i is out.。淘汰玩家给出的剩余数字将被忽略。如果在同一轮有超过一位的玩家被淘汰，则按他们的序号的升序输出。当游戏结束，在最后一行输出Winner(s): W1 W2 ... Wn，其中W1,W2,…,WnW_1,W_2,\dots,W_nW1​,W2​,…,Wn​是赢家的序号升序。所有数字打印在一行且必须由一个空格隔开，且不能有多余空格。如果没有赢家，输出No winner.。

# Idea

根据题意，需要记录的东西有

  1. 已经出现过的数字
  2. 前面已经出现过的数字的差

因为2很明显需要由1推得，所以1毫无疑问是用unordered_set存
然后2，我本来也是用unordered_set存的，但是一个测试点TLE了，后来发现可以用bool数组标记，就AC了

# Code

#include<bits/stdc++.h>
using namespace std;
const int N=16,M=1e3+7,K=1e5+7;
int a1,a2,n,m,f[N][M];
unordered_set<int>st;
bool dif[K],vis[N];
vector<int>p;
int res;
int main(){
    scanf("%d%d%d%d",&a1,&a2,&n,&m);
    for(int i=1;i<=n;i++){
        for(int j=1;j<=m;j++)
            scanf("%d",&f[i][j]);
    }
    dif[abs(a1-a2)]=1;
    st.insert(a1);
    st.insert(a2);
    res=n;
    // round
    for(int i=1;i<=m;i++){
        if(res==0)
            break;
        // people
        for(int j=1;j<=n;j++){
            if(res==0)
                break;
            if(vis[j])
                continue;
            if(!dif[f[j][i]]||st.count(f[j][i])){
                vis[j]=1;
                res--;
                cout<<"Round #"<<i<<": "<<j<<" is out.\n";
            }else{
                for(auto &k:st)
                    dif[abs(f[j][i]-k)]=1;
                st.insert(f[j][i]);
            }
        }
    }
    if(res==0){
        printf("No winner.\n");
    }else{
        printf("Winner(s):");
        for(int i=1;i<=n;i++)
            if(!vis[i])
                printf(" %d",i);
    }
    return 0;
}

# 7-3 Safari Park (25分)

关键原文及大意：

A safari park（野生动物园）has K species of animals, and is divided into N regions. The managers hope to spread the animals to all the regions, but not the same animals in the two neighboring regions.
野生动物园有KKK种动物，被分到NNN个区域。管理者希望将动物分配到所有区域，但相同动物不能在相邻区域。

For each plan, print in a line Yes if no animals in the two neighboring regions are the same, or No otherwise. However, if the number of species given in a plan is not K, you must print Error: Too many species. or Error: Too few species. according to the case.
对于每一种分配计划，如果没有两个相邻区域的动物是相同的，输出Yes，否则，输出No。但是，如果计划中给出的动物种类不是KKK，必须根据样例输出Error: Too many species.或Error: Too few species.。

# Idea

本来理解的题意是在Yes下去区分Too many还是Too few，所以中间写了一个剪枝，但发现测试用例并不是这样的，2 2 2 2 2 2的那个对不上
再看了一眼发现是先区分Too few，Too much还是刚好，再去刚好里区分Yes还是No，因为之前写的剪枝忘记删了，这会影响到不同动物的计数，所以去掉之后就AC了

# Code

#include<bits/stdc++.h>
using namespace std;
const int N=520;
int n,r,k,m;
bool g[N][N];
int main(){
    cin>>n>>r>>k;
    int u,v;
    for(int i=0;i<r;i++){
        cin>>u>>v;
        g[u][v]=g[v][u]=1;
    }
    cin>>m;
    while(m--){
        unordered_map<int,vector<int>>mp;
        bool f=1;
        for(int i=1;i<=n;i++){
            cin>>u;
            for(auto &j:mp[u]){
                if(g[i][j]){
                    f=0;
                    break;
                }
            }
            mp[u].push_back(i);
        }
        int cnt=mp.size();
        if(cnt==k){
            if(!f)
                cout<<"No\n";
            else
                cout<<"Yes\n";
        }else if(cnt<k)
            cout<<"Error: Too few species.\n";
        else 
            cout<<"Error: Too many species.\n";
    }
    return 0;
}

# 7-4 Replacement Selection (30分)

关键原文及大意：

Notice that as soon as the first record is written to an output tape, the memory it used becomes available for another record. Assume that we are sorting in ascending order, if the next record is not smaller than the record we have just output, then it can be included in the run.
注意到，只要第一条记录被写到输出的条带上，这条记录使用的存储空间就对其他记录可用。假设我们要按升序排序，如果下一个记录不小于我们刚刚输出的记录，这条记录就可以进入这一轮。

For example, suppose that we have a set of input { 81, 94, 11, 96, 12, 99, 35 }, and our memory can sort 3 records only. By the simplest method we will obtain three runs: { 11, 81, 94 }, { 12, 96, 99 } and { 35 }. According to the replacement selection algorithm, we would read and sort the first 3 records { 81, 94, 11 } and output 11 as the smallest one. Then one space is available so 96 is read in and will join the first run since it is larger than 11. Now we have { 81, 94, 96 }. After 81 is out, 12 comes in but it must belong to the next run since it is smaller than 81. Hence we have { 94, 96, 12 } where 12 will stay since it belongs to the next run. When 94 is out and 99 is in, since 99 is larger than 94, it must belong to the first run. Eventually we will obtain two runs: the first one contains { 11, 81, 94, 96, 99 } and the second one contains { 12, 35 }.
例如，假设我们输入一个集合{81,94,11,96,12,99,35}\{81, 94, 11, 96, 12, 99, 35\}{81,94,11,96,12,99,35}，我们的内存只能排序3条记录。通过最简单的方式，我们可以有三轮{11,81,94},{12,96,99},{35}\{ 11, 81, 94 \}, \{ 12, 96, 99 \}, \{ 35 \}{11,81,94},{12,96,99},{35}。根据替换选择排序，我们将读入并排序最开头的3条记录，{81,94,11}\{ 81, 94, 11 \}{81,94,11}，111111作为最小的数字输出。然后一个空间可用，所以969696被读进来，并加入第一轮，因为它大于111111。现在我们有{81,94,96}\{ 81, 94, 96 \}{81,94,96}。在818181输出之后，121212进来了，但是它必须属于下一轮，因为它小于818181。因此我们有{94,96,12}\{ 94, 96, 12 \}{94,96,12}，但是121212将停留在内存中，因为它属于下一轮。当969696输出999999进来之后，因为999999大于949494，所以999999必须属于第一轮。事实上，我们将拥有两轮：第一轮{11,81,94,96,99}\{ 11, 81, 94, 96, 99 \}{11,81,94,96,99}和第二轮{12,35}\{12,35\}{12,35}。

For each test case, print in each line a run (in ascending order) generated by the replacement selection algorithm. All the numbers in a line must be separated by exactly 1 space, and there must be no extra space at the beginning or the end of the line.
对于每个测试样例，按轮数升序，一行打印一轮。一行的每个数字间用一个空格隔开，且在首或尾不能有多余空格。

# Idea

我的24分是因为题意理解错了（没有好好看例子）
错误的题意理解：当优先队列满的时候才输出最小元素
正确的题意理解：每轮都输出最小元素，直到优先队列为空，剩下的都归到第二轮
这里的priority_queue存的是当前run，next run我用的vector存

看到{96,99,12}\{96,99,12\}{96,99,12}才反应过来是这样，12是占存储空间的（不在优先队列里，但在内存中），这样如果存储空间被mmm个第二轮的占满的话，后面的都参与不到第一轮，因为没有空间将他们读进来了，即使后面有100100100，那100100100也不会在第一轮输出
如果是按上面24分那种理解的话，应该是{96,99},{12}\{96,99\},\{12\}{96,99},{12}，12是不占存储空间的，这样后面的肯定都能读完，如果后面有100100100，那么100100100肯定能在第一轮输出，但这和正确答案是不一样的

# Code

#include<bits/stdc++.h>
using namespace std;
typedef long long ll;
int n,m,pos;
ll pre;
bool f=0;
priority_queue<ll,vector<ll>,greater<ll>>q;
vector<ll>a,v;
int main(){
    cin>>n>>m;
    ll num;
    for(int i=0;i<n;i++){
        cin>>num;
        a.push_back(num);
    }
    while(!a.empty()){
        v.clear();
        f=0;
        pos=min(m,n);
        for(int i=0;i<pos;i++)
            q.push(a[i]);
        for(int i=pos;i<n;i++){
            if(q.empty()){
                for(int j=i;j<n;j++){
                    v.push_back(a[j]);
                }
                break;
            }
            pre=q.top();
            q.pop();
            if(f)
                cout<<" ";
            cout<<pre;
            f=1;
            if(a[i]>=pre){
                q.push(a[i]);
            }else{
                v.push_back(a[i]);
            }
        }
        while(!q.empty()){
            pre=q.top();
            q.pop();
            if(f)
                cout<<" ";
            cout<<pre;
            f=1;
        }
        n=v.size();
        a.resize(n);
        for(int i=0;i<n;i++)
            a[i]=v[i];
        cout<<'\n';
    }
    return 0;
}
