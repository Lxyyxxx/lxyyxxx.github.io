# 键和超键

# 键

  1. A1,...AnA_1,...A_nA1​,...An​决定其他所有属性
  2. A1,...An{A_1,...A_n}A1​,...An​真子集中，没有一个可以决定其他所有属性（最小）

# 超键

包括键，但不一定最小

# 平凡FD

关系上的一个约束对所有关系示例都成立，且与其他约束无关

# 闭包

{A1,...An}+\{A_1,...A_n\}^{+}{A1​,...An​}+

  1. 初始结果X为{A1,...An}\{A_1,...A_n\}{A1​,...An​}
  2. 反复找FD：B1,...Bn−>CB_1,...B_n -> CB1​,...Bn​−>C，使B1,...BnB_1,...B_nB1​,...Bn​在X中，而CCC不在X中，把CCC加入X
  3. 没有新元素加入X时结束

# 传递规则

FD：A1,...An−>B1,...,BmA_1,...A_n -> B_1,...,B_mA1​,...An​−>B1​,...,Bm​和FD：B1,...,Bm−>C1,...,CkB_1,...,B_m -> C_1,...,C_kB1​,...,Bm​−>C1​,...,Ck​成立，则有FD：A1,...An−>C1,...,CkA_1,...A_n -> C_1,...,C_kA1​,...An​−>C1​,...,Ck​成立

# 基本集

和FD集合S等价的FD集合都是S的基本集

# 最小基本集

  1. B中所有FD右边都是单一属性
  2. 从B中删除任一FD，不再是基本集
  3. B中任一FD，从左边删除属性，B不再是基本集

# Armstrong公理

  1. 自反律
  2. 增广律
  3. 传递律

# FD投影算法

  1. 初始化结果T为空
  2. 对投影后每一个子集计算闭包，将所有非平凡FD加入T
  3. T为FD基本集，需要构造最小化基本集

# Boyce-Codd范式 BCNF

左边是超键的平凡FD
不能保持函数依赖，但能消除FD引起的冗余

  1. 关系是BCNF直接返回结果
  2. 存在BCNF违例X−>YX -> YX−>Y，选择R1=X+R_1=X^{+}R1​=X+作为一个关系，R2R_2R2​包含XXX及不在X+X^{+}X+中的属性
  3. 计算R1R_1R1​的FDS1S_1S1​、R2R_2R2​的FDS2S_2S2​
  4. 递归分解S1S_1S1​、S2S_2S2​

# 无损连接

分解到的关系进行自然连接可以准确恢复原始关系

# 第三范式 3NF

左边仅由超键或右边仅由主属性构成的非平凡FD
保持函数依赖，但不能消除FD引起的冗余

  1. 找FD的一个最小基本集G
  2. G中每个FDX−>AX -> AX−>A，将XA作为分解出的某个关系的模式
  3. 分解出的关系模式若均不包含R的超键，则增加一个关系（包含R的任何一个键）

# 多值依赖 MVD

关系中有两个属性集的值以所有可能的组合方式出现，两个属性或属性集合间相互独立
A1,...An−>−>B1,...,BmA_1,...A_n ->-> B_1,...,B_mA1​,...An​−>−>B1​,...,Bm​

# 第四范式 4NF

左边是超键FD的MVD
可以消除MVD带来的冗余，但不保持FD和MVD

  1. 令关系R=R0R=R_0R=R0​，FD和MVD集合S=S0S=S_0S=S0​
  2. 在RRR中找4NF违例，记为A1,...An−>−>B1,...,BmA_1,...A_n ->-> B_1,...,B_mA1​,...An​−>−>B1​,...,Bm​
  3. 将RRR分解为模式是A和B的R1R_1R1​和模式是A及R中不属于A和B的其他属性的R2R_2R2​
  4. 找出在R1R_1R1​、R2R_2R2​上成立的FD、MVD，进行递归分解

4NF包含于BCNF包含于3NF

设元组t在包R中出现n次，在包S中出现m次

  1. 并 R⋃SR\bigcup SR⋃S   t出现n+mn+mn+m次
  2. 交 R⋂SR\bigcap SR⋂S   t出现min⁡(m,n)\min (m,n)min(m,n)次
  3. 差 R−SR-SR−S   t出现max⁡(0,n−m)\max (0,n-m)max(0,n−m)次
  4. 投影 π\piπ
  5. 选择 σ\sigmaσ
  6. 笛卡尔积 R×SR\times SR×S   rs出现mnmnmn次
  7. 自然连接 R⋈SR\bowtie SR⋈S
  8. θ\thetaθ连接

  3. 分组操作   γL(R)\gamma_L (R)γL​(R)   L为元素列表
  4. 扩展操作   πL(R)\pi_L (R)πL​(R)
  5. 排序算子   τL(R)\tau_L (R)τL​(R)

# 关系代数与Datalog

  1. 并 R⋃SR\bigcup SR⋃S
    规则1：Answer(a1,…,an)<-R(a1,…,an)
    规则2：Answer(a1,…,an)<-S(a1,…,an)
  2. 交 R⋂SR\bigcap SR⋂S
    Answer(a1,…,an)<-R(a1,…,an) AND S(a1,…,an)
  3. 差 R−SR-SR−S
    Answer(a1,…,an)<-R(a1,…,an) AND NOT S(a1,…,an)
  4. 投影 π\piπ
    Answer(a1,a2)<-R(a1,…,an)
  5. 选择 σ\sigmaσ
    比较复杂
    连接AND Answer(a1,…,an)<-R(a1,…,an) AND 条件
    连接OR
    Answer(a1,…,an)<-R(a1,…,an) AND 条件
  6. 笛卡尔积 R×SR\times SR×S
    Answer(a,b,c,x,y,z)<-R(a,b,c) AND S(x,y,z)
  7. 自然连接 R⋈SR\bowtie SR⋈S
    Answer(a,b,c,d)<-R(a,b,c) AND S(b,c,d)
  8. θ\thetaθ连接  
    Answer(a,rb,rc,sb,sc,d)<-R(a,rb,rc) AND S(sb,sc,d) AND 条件
